# Mary — Personal Assistant by Finoveo

A smart personal assistant that integrates with Google Calendar and Gmail to keep you organized.

## Features
- **Daily Briefing** — Calendar events, work emails, and a daily Bible verse
- **Task Management** — Add, complete, and track tasks with priorities
- **Push Notifications** — Reminders, 10-min meeting alerts, 7am morning & 8pm evening nudges
- **Chat Interface** — Natural language interaction with calendar and email
- **Tomorrow's Preview** — Evening summary of the next day's schedule
- **Shareable Bible Verse** — Daily verse from the Catholic/Orthodox tradition

## Setup

```bash
npm install
npm run dev
```

## Deploy to Vercel

1. Push this repo to GitHub
2. Go to vercel.com → New Project → Import from GitHub
3. Vercel auto-detects Vite — click Deploy
4. Your app is live at your-project.vercel.app

## Add to Home Screen (PWA)
- **iPhone**: Open in Safari → Share → Add to Home Screen
- **Android**: Open in Chrome → Menu → Add to Home Screen
